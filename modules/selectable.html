tbot.selectable¶

Warning

The tbot.selectable module is deprecated in favor of the new “tbot context” mechanism. Please read the Context guide for an introduction.

The tbot.selectable modules gets filled with the machines from selected configuration at startup. The following placeholders exist (which will be replaced by proper machines):

tbot.selectable.LabHost¶

alias of LocalLabHost

class tbot.selectable.UBootMachine(lab: LocalLabHost, *args: Any)[source]¶

Bases: UBootShell, ContextManager

Dummy type that will be replaced by the actual selected U-Boot machine at runtime.
